# Smart-banking-management-system
A Python-based Smart Banking Management System using file handling and OOP concepts
A command-line based Bank Management System built using Python.  
This application simulates basic banking operations such as account creation, deposits, withdrawals, and transaction tracking with persistent data storage.
##  Project Overview
The Smart Bank Management System is a Python-based CLI application designed to demonstrate core programming concepts and object-oriented principles.  
All account data is securely stored in a JSON file, ensuring data persistence between sessions.
This project is ideal for showcasing foundational Python development skills in a real-world scenario.
## Concepts Demonstrated
This project covers:
Variables & Data Types
Lists and Dictionaries
Conditional Statements
Loops
Functions
Classes & Objects
JSON File Handling
Exception Handling
CLI Application Design
##  Features
- Create new bank accounts
- Deposit money
- Withdraw money
- View account details
- View transaction history
- Delete accounts
- Persistent storage using JSON
- Input validation and exception handling
## 🛠 Technologies Used
- Python 3
- JSON (for data storage)
- Object-Oriented Programming (OOP)
- File Handling
- Exception Handling
